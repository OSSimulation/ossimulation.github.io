NEWS


Builds are now available for iOS 6 Simulator! Builds will be released every time a commit is pushed. However, these nightly builds will only be downloadable if you have a GitHub account. Since we can't remove this requirement, we also release stable builds less frequently, so anyone can download the simulator.

Due to a security vunerability found in all Unity versions 2017.1+, we've updated our Unity version to 2021.3.45f2 to mitigate it. As such, you'll need to use that version to edit the iOS 6 Simulator and the OS Menu. Please update your forks as soon as possible.

After nearly a year of hiatus, a new update for the iOS 6 Simulator has been released. An update showcase showing off the new update has been released on our YouTube channel, and a new devlog has been uploaded on Tailspin96's channel.
