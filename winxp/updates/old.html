Build 2701 (Windows XP Gui Test 2.0 Beta 4)

Published: 1 March, 2024

After almost a year of no updates, an update has been released. Here are the changes:

•

Fixed bug where the world would not regenerate
•

Fixed an issue where avatar colours would not apply to your in-game Roblox avatar

•

Re-added the loading screen for Roblox viewport edition
•

Added a "logs" command to the command prompt that enables or disables logging of errors to the command prompt
•

Moved the OS Menu to the right as the Roblox menu has been changed
•

Added the option to disable the OS Menu due to it's new location
•

Shutdown has been (finally) added
•

Added a "badgehax" command that unlocks all badges. This was added due to the amount of unobtainable badges in the game
•

Fixed mouse position being off centered in Roblox's fullscreen mode
•

The Luigi background has been replaced due to Nintendo taking it down

Build 2482

Published: 29 November, 2022
•

Added the Blue Screen of Death! You can activate it by typing in "del" in the Command Prompt or by clicking on the "Blue Screen of Death" Achievement 10 times in the OS menu. When a BSoD is about to occur, the operating system begins to corrupt, then begins to delete every file. and then it crashes. You have to rejoin the game once the BSoD appears to continue playing. Because of that, the Close Menu option and the ESC buttons are disabled in the OS Menu as the Windows XP files have been deleted
